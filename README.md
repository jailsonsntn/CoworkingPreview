# Coworking Saúde Therapi - Landing Page

Landing page institucional focada em geração de leads e agendamento de visitas para o Coworking Saúde Therapi, em Teresina-PI.

## Stack
- HTML5
- CSS3
- JavaScript Vanilla
- Deploy estático (Vercel)

## Recursos implementados
- Hero com vídeo de fundo
- Modal com vídeo de apresentação
- Galeria com lightbox
- Scroll animations
- Botões premium estilo vidro
- WhatsApp flutuante e CTA estratégicos
- Mapa com card de localização

## Deploy na Vercel
1. Importar o repositório no painel da Vercel.
2. Framework Preset: Other.
3. Build Command: vazio.
4. Output Directory: .
5. Deploy.

## Configuração Google (pós-deploy)
1. Google Search Console: adicionar propriedade do domínio.
2. Trocar o valor de google-site-verification no head.
3. Enviar sitemap: https://coworkingsaudetherapi.com.br/sitemap.xml
4. Solicitar indexação da homepage.

## Analytics e Pixel
